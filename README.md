# COCO-API

The back-end for product-compare-angular.


### Prerequisites

* Java 8
* maven

Install Java 8 and maven using [brew](https://brew.sh/) (on mac)

```sh
$ brew install java8
$ brew install maven
```
### Steps to build

```
$ mvn clean install
```

### Steps to run 

```
$ mvn spring-boot:run
```

### Successful Run
Browse to localhost:8080 and should get a valid response
